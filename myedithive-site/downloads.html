If macOS warns about an unidentified developer

Right-click the .pkg file, choose Open, then click Open again in the dialogue that appears. This is normal for unsigned beta builds and only happens once.

First launch sets up in seconds

From Beta 1.4.3 the analysis engine is bundled inside the installer, so the first launch sets up in a few seconds with no download — it works fully offline.
